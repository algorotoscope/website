{%- comment -%} Returns a thumbnail URL for a work. Prefers YouTube thumbnail (derived from work.video) because the S3-hosted image/square URLs are not publicly readable. Falls back to square/image when no video is present. Usage: {% capture thumb %}{% include work-thumb-url.html work=work %}{% endcapture %} {%- endcomment -%} {%- if include.work.video -%} {%- assign vid = include.work.video | split: '/embed/' | last | split: '?' | first | split: '&' | first -%} https://i.ytimg.com/vi/{{ vid }}/hqdefault.jpg {%- elsif include.work.square -%} {{ include.work.square }} {%- elsif include.work.image -%} {{ include.work.image }} {%- endif -%}
